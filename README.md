# miw planner

A calm, high-performance weekly planner for projects, tasks, and goals.

## Features
- **Project & Task Management**: Keep tasks structured and organized.
- **Supabase Cloud Syncing**: Live two-way synchronization between your local state and the cloud.
- **Premium Performance Dashboard**: Insights, analytics, and completions.
- **Project Sharing**: Generate base64 shareable URLs to share project templates seamlessly.
- **Customizable Themes & Textures**: Beautiful, responsive layout with various newsreader styling options.
